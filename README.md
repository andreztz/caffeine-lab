# caffeine-lab
> Inhibits the screensaver and sleep mode like a boos.


## Installation

OS X & Linux & Windows:

```sh
$ pip install --user git+https://github.com/andreztz/caffeine-lab.git
```

## Usage example

```sh
$ caffeine-lab
```


## Development setup

```sh
$ git clone https://github.com/andreztz/caffeine-lab.git
$ poetry install
```
